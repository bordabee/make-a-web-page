Lesson 1: The Basics of the Web and HTML

Introduction to the Web

The World Wide Web is a collection of HTML documents.
HTML stands for HyperText Markup Language, this is the basis for the majority of web pages.
HTML is what connects most types of documents together through hyperlinks. HTML can be made up of text content (what you see), markup (arrangement of content), references to other documents, and links to other pages.

Computers are Stupid

Programmers need to write in a language a computer understands; this way a computer can follow the instructions the programmer wrote.
Therefore, in order to be precised, it is imperative to use the right syntax.

HTML Attributes

Anchor tags: adds a link to a text or word. When clicked, it directs you to another page.
Image tags: adds a link to an image. Another page is opened when the image is linked.
Whitespace: text on multiple lines using the break tag (void tag) or the paragraph tag (not a void tag).
Inline/Block: an inline tag tells when a line ends, while a block tag creates an invisible box. Examples of inline tags include the anchor, span, break, image, and strong tags. While the paragraph and div tags are types of block tags.

Document Structure

The basic structure of an HTML document includes:
a DOCTYPE in the first line,
an opening HTML tag on the second line,
an opening and closing head tags, which will include title tags,
an opening an closing body tags, which will include the page's content,
and a closing html tag.

Lesson 2: Creating a Structured Document with HTML

Developer Tools

Using a browser's developer tools, one is able to see then structure of a website's page.
It is worth noting that not all elements in the structure are visible in the webpage, yet all elements are rectangular.

Tree-like Structure

The browser buildes a tree-like structure of the page. Each sideways triangle has its own opening and closing tags.

Boxes, Everywhere!

Everything in HTML is a box, regardless of what shape it shows in the browser.
This box can be styled into different colors and shapes.
Use the div tag to create boxes.

The Difference Between Code Editors

A code editor is a program designed to help programmers edit code.
At this level there are three code editors being compared: Scratchpad, CodePen, and Sublime.
Scratchpad is recommended for writing code, and CodePen for sharing with others.
